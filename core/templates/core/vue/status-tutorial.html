Cool, how do I use it?

If you work on a wetlab located in 3rd floor, just press

WETLAB

located on left menu. This should load all the analyses that are:
1. Completed within last 2 weeks.
2. Lane was requested within last 2 months, but not yet completed.
3. Library has to be post-2019

There are also INCOMPLETE and NO ANALYSIS lists:

NO ANALYSIS: Get information from libraries with correct lane number but no analyses attached to it.
INCOMPLETE: Get information from all analyses with run_status not set as either one of complete, align_complete or hmmcopy_complete.

Or you can chose Project if you wants to have a customized collection of DLP libraries.

Excuse me, what is Requested Date and Lane?

Requested Date: How is it computed? Firstly, does the row have analysis attached to it? If yes, we look at all the sequencings related to that analysis and get the latest Lane Requested Date from sequencings. If no, we look at all the sequencings related to the library instead.
For WETLAB and NO ANALYSIS lists, second method is used as default.
Lane: You can see that Lane value is displayed as a fraction. Numerator represents the number of lanes physically attached to the analysis. Denominator is the sum of all the related sequencings' requested number of lanes.
