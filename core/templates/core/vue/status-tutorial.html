This page exists to help you track status of analyses.
We reckon that it is slightly inconvenient to have multiple pages open to track analyses, so we came up with this page to display all related information from different sources.


Cool, how do I use it?

If you work on a wetlab located in 3rd floor, just press located on left menu. This should load all the analyses that are:
1. Updated within last two weeks.
2. Related to sequencings with mismatching number of lanes to requested lane number.
3. Library has to be post-2019

But let's say you want to create your own list.
Then simply click add located in lower left corner.
This will take you to ProjectTag create page.
ProjectTag is a new model we came up with. It currently only has 2 fields: Title and samples. You set the title and add all the samples you want to attach, then the table will be populated with all the analyses attached to that samples.
Similarly, use remove to remove ProjectTag.
It does not auto refresh at the moment.
